## About

CloudFront is AWS' content delivery network for delivering data, video, applications, and APIs globally with low latency. The CloudFront REST API exposes operations for distributions, origins, cache behaviors, invalidations, origin access identities/control, key groups, CloudFront Functions, Lambda@Edge associations, real-time log configs and field-level encryption. Authentication uses AWS Signature v4.
